# YT Music Overlay

Electron desktop app with an embedded YouTube Music player and a transparent always-on-top lyrics overlay.

## Run

```powershell
npm install
npm start
```

Open YouTube Music inside the app, sign in, play a song, and switch to the Lyrics tab. The overlay reads only the lyrics already shown by YouTube Music.

The app also loads the unpacked Better Lyrics extension from `extensions/better-lyrics` into the embedded YouTube Music session.
